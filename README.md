# VGZ CMS

Magnolia light module for VGZ.

## Installation

```
mgnl jumpstart
cd light-modules
git clone https://github.com/aktion-hip/vgz-module.git
```

## Features

- Theme
- Pages
- Components: Text, People-Entry, Form (decorated Mangolia Form Module)
- App: VGZ-Events
